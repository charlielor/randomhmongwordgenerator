This is a random Hmong word generator (words generated may not be an actual Hmong word) base on these facts:
  • Monosyllabic - All Hmong words are one syllable
  • Algorithm to derive most Hmong RPA word: (consonant) + (vowel) + (tone)
  • Many consonants (single, double, triple, quad), not so many vowels (single, double), 8 tones
An underline (_) signifies that there is no letter(s) (empty)
